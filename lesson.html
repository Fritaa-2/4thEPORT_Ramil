My Lessons

Lesson: Loops

Loops allow repeating a block of code until a condition is met.

for (let i = 1; i <= 5; i++) { console.log("Number " + i); }

Lesson: Events

Events allow interaction with web pages.

document.getElementById("btn").addEventListener("click", function() { alert("Button clicked!"); });

Lesson: If-Else Statements

If-Else statements help make decisions in code.

let age = 20; if (age >= 18) { console.log("You are an adult."); } else { console.log("You are a minor."); }

Lesson: Switch Statements

Switch statements allow handling multiple conditions.

switch(day) { case 1: console.log("Monday"); break; case 2: console.log("Tuesday"); break; default: console.log("Other day"); }

Lesson: Functions

Functions are reusable blocks of code.

function greet(name) { return "Hello, " + name + "!"; }
